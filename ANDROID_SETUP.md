# 🤖 Telegram Bot - Android এ রান করার জন্য

## ⚡ Replit এ রান করুন (সবচেয়ে সহজ)

### স্টেপ ১: Replit এ যান
- https://replit.com এ যান
- সাইন আপ করুন (GitHub দিয়ে সাইন আপ করতে পারেন)

### স্টেপ ২: প্রজেক্ট ইমপোর্ট করুন
- **"+ Create"** বাটন ক্লিক করুন
- **"Import from GitHub"** নির্বাচন করুন
- এই URL পেস্ট করুন: `https://github.com/lamiyark595-prog/telegram-otp-bot.git`
- **"Import"** ক্লিক করুন

### স্টেপ ৩: রান করুন
- **"Run"** বাটন ক্লিক করুন
- কনসোলে দেখবেন: ✅ বট সফলভাবে চালু হয়েছে!

### স্টেপ ৪: বট ব্যবহার করুন
- টেলিগ্রামে যান: `@fast_otp_work_bot`
- `/start` লিখুন এবং পাঠান
- 3 টা বাটন দেখা যাবে!
