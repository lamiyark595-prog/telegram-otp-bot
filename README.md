# 🤖 Telegram Bot

একটি সাধারণ টেলিগ্রাম বট যেখানে /start করলে কিছু বাটন আসবে।

## 🚀 সেটআপ

### 1. রিকোয়ারমেন্ট ইনস্টল করুন

```bash
pip install -r requirements.txt
```

### 2. বট চালান

```bash
python main.py
```

### 3. টেলিগ্রামে /start দিন

- `@fast_otp_work_bot` সার্চ করুন
- `/start` কমান্ড দিন
- বাটন পাবেন
